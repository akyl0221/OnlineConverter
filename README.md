<h1>Web App to download and convert Youtube file to MP3</h1>
<h2>Install</h2>

<p>$ git clone https://github.com/akyl0221/OnlineConverter.git</p>
<p>$ virtualenv venv -p python3</p>
<p>$ source venv/bin/activate</p>

<h2>Requarements</h2>
<ul>
    <li>Python 3.7.3</li>
    <li>Youtube-dl</li>
    <li>Django v.2.2.2</li>
</ul>
